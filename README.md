## ðŸš€ Installation & Setup

Since this project was developed specifically for the **Eclipse IDE** environment, follow these steps for the best experience:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Shauwye/Bosque-Health.git](https://github.com/Shauwye/Bosque-Health.git)
    ```
2.  **Open Eclipse IDE.**
3.  **Import the project:**

    * Select `General` > `Existing Projects into Workspace`.
    * Select the `main` folder as the root directory.
4.  **Configure JDK:** Ensure your project build path is set to **Java 23**.
5.  **Run:** Locate the main entry point and run as a **Java Application**.
